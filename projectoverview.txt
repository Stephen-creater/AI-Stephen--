# 个人网站项目概述

## 项目目标
这是一个个人展示网站，包含首页、作品集、兴趣爱好、知识库和联系方式等多个页面，用户可以通过这个网站了解网站主人的个人信息、作品、兴趣和知识体系。

## 页面结构
1. **首页 (index.html)** - 网站主页，包含个人简介和各个板块的预览
2. **作品集 (portfolio.html)** - 展示个人项目和作品
3. **兴趣爱好 (interests.html)** - 展示个人兴趣，如电影、书籍等
4. **知识库 (knowledge.html)** - 展示个人知识体系和学习资源
5. **联系方式 (contact.html)** - 提供与网站主人的联系方式
6. **常见问题 (faq.html)** - 提供关于服务和合作的常见问题解答

## 技术栈
- HTML5 - 页面结构
- CSS3 - 样式和动画
  - TailwindCSS - 快速样式开发
  - 自定义CSS - 特殊样式和动画
- JavaScript - 交互和动态效果
  - 原生JavaScript - 无框架依赖

## 特色功能
### 1. 响应式设计
- 网站适配各种屏幕尺寸，从移动设备到桌面显示器
- 使用媒体查询和弹性布局实现

### 2. 交互式动画
- **鼠标光晕效果** - 跟随鼠标移动的紫色光晕，增强用户体验
- **滚动渐入动画** - 元素随着页面滚动逐渐显示
- **悬停动画** - 卡片、按钮等元素在鼠标悬停时的动态效果
- **首页文字动画** - 标题文字逐字显示动画，"无限可能"渐变色动画效果
- **动态背景渐变** - 首页背景使用渐变动画，提升视觉体验

### 3. 知识库功能
- **分类展示** - 将知识资源按主题分类展示
- **链接分享** - 每个知识卡片右上角的分享按钮可复制链接
- **视觉反馈** - 复制链接时显示提示信息

### 4. 联系表单功能
- **用户消息提交** - 通过表单发送名称、邮箱和消息
- **数据存储** - 使用localStorage存储表单提交数据
- **复制联系信息** - 一键复制邮箱、电话和微信ID
- **视觉反馈** - 复制成功时显示提示信息
- **二维码展示** - 展示微信二维码方便移动端用户添加联系人

### 5. 社交媒体整合
- 提供与各社交媒体平台的链接
- 悬停显示详细联系信息

### 6. **数据持久化存储**
   - 提交的表单数据以JSON格式保存
   - 文件命名格式为contact_姓名_时间戳.json
   - 数据包含姓名、邮箱、消息内容和提交时间
   - 保存路径为data/contacts/目录

### 7. **常见问题解答功能**
   - 提供可展开/收起的问答界面
   - 实现平滑的动画过渡效果
   - 问题按类别组织，便于用户查找
   - 渐入式加载提升用户体验
   - 响应式设计适配不同设备

### 8. **首页布局与视觉优化**
   - **居中对齐** - 首页内容采用居中对齐，使页面更加平衡
   - **增加留白** - 标题与副标题之间增加留白，使页面布局更加清晰
   - **文字逐字动画** - 标题文字逐字显示动画效果，增强视觉吸引力
   - **渐变色文字** - "无限可能"使用渐变色动画效果，突出重要信息
   - **背景动态效果** - 添加动态渐变背景，增强页面层次感
   - **光晕居中** - 背景光晕效果居中显示，视觉平衡感更强

## 主要功能详解

### 鼠标光晕效果
光晕效果通过在js/animations.js中创建两个跟随鼠标的元素实现：
- `cursor-follow` - 小型跟随光标
- `cursor-glow` - 较大的外部光晕
这两个元素使用requestAnimationFrame进行平滑动画，创造出流畅的跟随效果。

### 卡片悬停动画
为卡片元素添加以下效果：
- 当鼠标悬停时，卡片向上移动(-5px)
- 添加紫色阴影效果增强视觉层次感
- 背景色微调，提升对比度

### 链接分享功能
知识库页面中的每个卡片右上角有分享按钮：
1. 点击按钮自动复制当前卡片的链接到剪贴板
2. 显示"链接已复制!"提示
3. 提示2秒后自动消失

### 联系表单功能
联系页面实现了以下功能：
1. **表单提交处理**
   - 用户填写名称、邮箱和消息后提交
   - 表单数据通过JavaScript收集并存储到localStorage
   - 同时生成JSON文件保存到data/contacts/目录
   - 提交成功后显示提示信息并重置表单

2. **联系信息一键复制**
   - 点击邮箱、电话、微信ID旁的复制按钮可快速复制内容
   - 复制成功后显示简洁的成功提示
   - 提供视觉反馈增强用户体验

3. **二维码展示**
   - 页面底部展示微信二维码
   - 提供视觉引导让用户扫码添加联系人

4. **表单验证**
   - 使用HTML5原生表单验证
   - 必填字段验证及电子邮箱格式验证

5. **数据持久化存储**
   - 提交的表单数据以JSON格式保存
   - 文件命名格式为contact_姓名_时间戳.json
   - 数据包含姓名、邮箱、消息内容和提交时间
   - 保存路径为data/contacts/目录

6. **常见问题解答功能**
   - 提供可展开/收起的问答界面
   - 实现平滑的动画过渡效果
   - 问题按类别组织，便于用户查找
   - 渐入式加载提升用户体验
   - 响应式设计适配不同设备

### 首页优化功能
1. **内容居中对齐**
   - 首页文字内容采用居中对齐布局
   - 按钮居中排列，视觉效果更加平衡

2. **标题文字动画**
   - 标题文字使用逐字显示动画效果
   - 每个字符独立动画，创造连续渐入效果

3. **"无限可能"文字特效**
   - 使用渐变色和动画效果强调关键词
   - 文字渐变色不断流动，创造动态视觉效果

4. **动态背景渐变**
   - 添加渐变背景随时间变化
   - 使用紫色系渐变增强科技感氛围

5. **增强留白和间距**
   - 增加标题与副标题间的留白，使内容更易阅读
   - 优化各元素间的间距，创造更和谐的视觉体验

6. **布局与结构优化**
   - 删除欢迎语，突出主标题
   - 将"探索、创造与无限可能"放在同一行，更加简洁有力
   - 背景光晕居中显示，增强整体平衡感
   - 调整段落宽度和行高，提升可读性
